The main purpose of this app is to automate adding tasks to user stories in Azure DevOp.

To do this a comma-separated list of user story IDs is needed. Click the download icon to pull IDs from a sprint. Alternatively you can just type or paste some IDs in, even straight from Excel.

This was designed to run within a company's protected area. DevOps users should not be encourgaed to paste their PATs in a public website.
