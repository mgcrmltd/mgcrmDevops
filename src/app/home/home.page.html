The main purpose of this app is to automate adding tasks to user stories in Azure DevOps.

To do this a comma-separated list of user story IDs is needed. Click the download icon to pull IDs from a sprint. Alternatively you can just type or paste some IDs in, even straight from Excel.

This site absolutely does not make any attempt to store the credentials anywhere. The code is available on GitHub, so it can be easily built and hosted locally.

The Hours page allows you to create groups, for which the hours on tasks will be summed. Within each group you specify text tags. It will add the hours for tasks starting with the tag text to that group.

This is just a utility. It hasn't been tested well, probably goes weird if you resize the browser and follows no UI guidelines. If you don't like it, just create your tasks manually.
